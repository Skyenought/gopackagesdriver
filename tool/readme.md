## Config

### args

- `keyword`: [env](#env)
- `merge`: [merge](#env)
- `pattern`: 用于指定含有需要生成 import json 的`文件目录`

### env

- `CWGO_GOPACKAGESDRIVER_KEYWORD`: "约定的生成文件目录中的提速关键字, 用于辨识需要生成的 import json"
- `CWGO_GOPACKAGESDRIVER_MERGE`: "用于将 `pattern` 目录下的所有含有 import json 汇总至 merge import json 的文件目录"
